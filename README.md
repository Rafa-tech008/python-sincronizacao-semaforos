# Python - Sincronização com Semáforos

## 📖 Objetivo

Aplicar os conceitos de sincronização entre processos e threads utilizando semáforos em Python.

## 📂 Conteúdo

O repositório contém exercícios sobre:

- Semáforos;
- Sincronização de processos;
- Sincronização de threads;
- Controle de acesso a recursos compartilhados.

## 💻 Tecnologias

- Python 3

## ▶️ Como executar

1. Instale o Python 3 em seu computador.
2. Clone este repositório ou faça o download dos arquivos.
3. Execute o arquivo desejado pelo terminal:

```bash
python nome_do_arquivo.py
```
